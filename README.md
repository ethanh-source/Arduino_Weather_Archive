# Smart Home Arduino Project (Archived)

An early embedded systems project developed during my time at Nanyang Polytechnic, focusing on basic IoT communication and microcontroller-based system integration using Arduino.

The project includes:
- Arduino source code
- Basic smart home control logic
- Demonstration video of prototype system behavior (link below)
  
Uses 2 Arduinos (1 RX, 1 TX)

Note: This is a legacy project and is not actively maintained.
